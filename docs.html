01. Installation

To install Ronin on your own VPS (Ubuntu 20.04+), run this command. Do not run this as root unless you want to live dangerously.

curl -s https://get.ronin.dev/install.sh | bash

Once installed, the dashboard will be available at port 3000.


Requirements

  • 2GB RAM
  • 2 CPU Cores
  • A burning desire for simplicity
